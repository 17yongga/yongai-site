YongAI Logo Package
===================

Source
------
Generated from the approved black YongAI logo direction: a restrained, consulting-firm style Y monogram with the spaced YONGAI wordmark.

Included PNG folders
--------------------
PNG/transparent
- Primary vertical lockup: black + white, transparent background, 4000px wide.
- Monogram mark: black + white, transparent background, 3000px wide.
- Wordmark: black + white, transparent background, 3000px wide.
- Optional horizontal lockup: black + white, transparent background, 4000px wide.

PNG/background
- Primary vertical lockup on white, ivory, and dark charcoal backgrounds.
- Monogram mark on white, ivory, and dark charcoal backgrounds.

PNG/web
- Website header variants: 2400x600.
- LinkedIn/banner variants: 1584x396.

PNG/favicon
- Monogram favicon/app icon PNGs from 16px through 1024px.
- favicon .ico with common favicon sizes.

Design meaning
--------------
- The single abstract Y mark is the primary symbol, avoiding separate decorative icons.
- The split diagonal cut through the Y suggests a controlled workflow path and decision gate.
- The upper-right separated stroke implies an AI-assisted branch that remains governed by the core operator workflow.
- The heavy serif-like Y form gives the mark an advisory, law-firm, and institutional feel rather than a typical SaaS/startup feel.
- The black/white system keeps the identity scalable, premium, and usable across light and dark mode.

Color system
------------
Black: #000000
White: #FFFFFF
Dark charcoal background: #0B0C0E
Warm ivory background: #FAF8F2

Usage notes
-----------
Use the transparent PNGs when placing the logo on a controlled background.
Use the dark-mode assets when the background is near-black or charcoal.
Use the monogram-only assets for favicons, profile images, app icons, and small placements.
Use the full vertical lockup for business cards, covers, presentations, and high-touch brand moments.
Use the horizontal header lockup for website navigation where vertical space is limited.


SVG and PDF additions
---------------------
This updated package adds SVG and PDF versions of the main logo assets. The web/header lockups use the selected tighter spacing option, where the YONGAI wordmark sits closer to the monogram.
Because the approved source was a high-resolution PNG rather than a native design file, the SVG/PDF files are vector-traced from the approved PNG artwork. Use the PNG files for exact pixel parity and the SVG/PDF files for scalable web, print, and layout placement.
